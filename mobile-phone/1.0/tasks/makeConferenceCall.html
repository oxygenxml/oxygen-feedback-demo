Setting up a conference call

You can easily set up a conference call with your family, friends, co-workers. Call the first participant and then join calls together as described in the following procedure.

Before starting, make sure that your SIM card is enabled for conference calls service.

  1. Call the first participant. See Making a call
  2. During the call, press key to add a participant. In this way the first participant in put on hold.
  3. Type the participant phone number or search for in contacts.
  4. Press the key.
  5. To join also the first contact to the conference call, press the key again.
  6. To join other contact to the conference, repeat steps 2 - 5.
